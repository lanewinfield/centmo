Centmo
=====

Pay people by the penny.

```sh
venmo pay @lanewinfield 23.19 "I hate you"
```

Setup
-----
Set up centmo by running:

```sh
centmo configure

> Venmo email [None]: brian@brianmoore.com
> Venmo password [None]:
> Verification code: 696969  # for 2 factor authentication
```

That's it!
